# Portfolio Profile

A dark minimalist portfolio built with React, TypeScript, and Vite.

The site presents Prananda Yoga as a backend-focused developer, with pages for projects, skills, profile information, and contact links. The UI uses a terminal-inspired black-and-white style with technology icons for tools, languages, and contact methods.

## Tech Stack

- React
- TypeScript
- Vite
- React Router
- Vitest
- Lucide React
- React Simple Icons

## Run Locally

```bash
npm install
npm run dev
```

## Useful Commands

```bash
npm run test:run
npm run lint
npm run build
```

## Edit Content

- `src/data/profile.ts` - profile, bio, and contact links
- `src/data/projects.ts` - project data
- `src/data/skills.ts` - skills and tools
